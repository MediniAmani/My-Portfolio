# Amani Medini Portfolio

Vite + React + TypeScript portfolio inspired by rachelhow.com’s minimal typography-first layout.

## Scripts

- `npm run build` - production build
- `npm run preview` - preview production build
- `npm run dev` - local development server

## Pages

- `/` Home
- `/work` Projects
- `/work/:slug` Case studies
- `/about` Bio, Now, timeline, speaking
- `/contact` Contact + CV downloads

CV files live in `public/cv/`.
